# 🧠 Programação Competitiva

Repositório com exercícios e soluções de **programação competitiva**, utilizados para treinamento em competições de algoritmos.

Os exercícios incluem problemas resolvidos durante:

* **Treinos para a OBI**
* **Olimpíada Brasileira de Informática (OBI)**
* **Maratona de Programação do CPS**

---

## 🏆 Competições

### 🇧🇷 OBI — Olimpíada Brasileira de Informática

A **OBI** é uma competição nacional de programação e raciocínio lógico voltada para estudantes.

Neste repositório estão algumas soluções desenvolvidas durante:

* Treinamentos
* Participações na competição

---

### 🏫 Maratona de Programação do CPS

Problemas resolvidos durante a **Maratona de Programação organizada pelo Centro Paula Souza (CPS)**.

Esses exercícios focam em:

* Lógica de programação
* Algoritmos
* Resolução de problemas em tempo limitado

---

## 📂 Estrutura do Repositório

Os exercícios estão organizados em pastas conforme:

* Treinos
* Competições
* Listas de exercícios

Cada arquivo contém a **implementação da solução para um problema específico**.

---

## 🛠 Linguagens Utilizadas

Dependendo do exercício, podem ser utilizadas linguagens como:

* **Java**
* **C**
* **C++**
* **Python**

---

## 🎯 Objetivo

Este repositório serve como:

* Registro de soluções de problemas de programação competitiva
* Prática de **algoritmos e estruturas de dados**
* Preparação para **competições de programação**
